# Pasarela de pago para compra de cryptomoneda

1. Crear archivo .env
2. Definir las variables de entorno segun el archivo: ."env.example"
3. Correr el contenedor
```
docker compose up -d
```
4. Ejecutar el programa
```
dotnet run 
```
